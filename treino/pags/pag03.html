Utilizando Links

Links Externos

Para a criação de links/âncoras é atráves da tag A, e um de seu atributos o HREF, é o que cria uma referência hypertexto.
Junto pode ser usado atributo target, para poder controlar o destino de abertura do site.
Segue os valores suportados:

  • _blank
    • Abre o link em uma nova janela em branco.
  • _self
    • Abre o link na janela ou frame atual (padrão).
  • _top
    • Vai desfazer todos os frames e abrir o destino no navegador completo.
  • _parent
    • Similar ao uso do _top em uma referência a janela mãe.
  • nome-do-arquivo
    • Caso esteja usando frames, indicar o nome da janela a abrir.

Um exemplo a seguir de um link/hypertexto:

Acesse o Youtube e descubra um mundo cheio de conteúdos!

Para saber mais sobre Dragon Ball, acesse a Wiki do anime!

Links Internos

É para a criação de mais páginas para o site, tendo entendimento de tais páginas estejam nos arquivos do seu servidor. Segue o exemplo:

Apertando aqui, você vai para a página teste 1.

É possível a utilização de um arquivo para armazenamento desses arquivos, exemplo a seguir:

Para acessar a página teste 2, clique aqui!

Nota:./ é para referir a pasta atual do servidor. (Antes do nome do arquivo)
../ é para voltar um nível para a pasta mais externa.

Links para Download

Para a criação de um link para download, é utilizado além da tag A, o atributo DOWNLOAD, configurado com o nome do arquivo a ser baixado.
E o atributo TYPE, para indicar ao navegador o tipo de arquivo que está sendo baixado. Segue abaixo os valores bastante usados nesse atributo:

  • application/zip
  • text/html
  • text/css
  • text/javascript
  • video/mp4
  • video/H264
  • video/JPEG
  • audio/aac
  • audio/mpeg
  • font/ttf
  • image/jpeg
  • image/png

Segue um exemplo de link de donwload:

Clique aqui para baixar o arquivo zip do livro.

Para colocar áudios, é utilizado a tag AUDIO. Segue um áudio a seguir:

É bom falar sobre os atributos principais da tag AUDIO.

  • Preload

    • Indica se o áudio será pré-carregado ou não. Ele aceita três valores:

    • Metadata

      • Carrega apenas as informações sobre o arquivo (tamanho, tempo, informações de direitos, etc)

    • None

      • Não vai carregar nada até que o usuário clique no botão play ou um script inicie a reprodução.

    • Auto(Padrão)

      • Vai carregar o arquivo de áudio inteiro assim que a página for carregada, mesmo que o usuário nunca aperte o play.

  • Controls

    • Vai apresentar o player na tela. Nota: Caso não seja colocado na tag <audio>, o controle será transparente e o usuário não poderá interagir com ele.

  • Autoplay

    • Inícia a reprodução de áudio assim que página carregar.

  • Loop

    • Faz com que o áudio seja repetido automaticamente assim que terminar a sua reprodução
