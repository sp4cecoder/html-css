Colocando Vídeos em Hospedagem própria

Para isso, utiliza-se a tag VIDEO, tem o mesmo funcionamento que a tag IMG para selecionar o arquivo. Vale falar sobre alguns atributos importantes dele:

  • Width

    • Vai indicar a largura que o vídeo vai ter na tela.

  • Poster

    • Configura uma imagem que vai aparecer como uma capa, em quanto o visitante não aperta o play para reproduzir o vídeo.

  • Controls

    • Vai configurar se os controles vão aparecer na parte inferior da mídia. Por padrão, os controles não aparecerão, mas basta colocar o atributo na tag <video>.

  • Autoplay

    • Diz para o navegador se o video vai começar a tocar automaticamente, assim que a página for carregada.

Vale resaltar de formatos suportados pelos navegadores: MPEG, WEBM e OGG, mas os dois primeiros são os que mais possuem compatibilidade com os navegadores.

Navegadores e formatos de video suportados

  • Microsoft Edge - .mp4 / .m4v
  • Apple Safira - .mp4 / .m4v
  • Google Chrome - .mp4 / .m4v / .webm / .ogv
  • Mozilla Firefox - .webm / .ogv
  • Opera - .webm / .ogv

Incorporando Vídeos Externos

Hospedar video é bastante, mas existe duas opções para resolver esse problema: O Youtube e o Vimeo

  • Youtube

    • Pode hospedar videos de forma gratuita e possui servidores bastante rápidos. Mas caso queira limitar a quantidade de pessoas para ver algum video, não será possível, já que o vídeo ficaria público.

  • Vimeo

    • Pode hospedar videos, mas de forma paga, diferente do Youtube, você consegue limitar quantar pessoas podem ver algum video, mas o player dele pode dar alguns travamentos.

Para poder inserir um vídeo externo, usaremos o Youtube como exemplo, você seleciona o video que deseja, clica em COMPARTILHAR, e em seguida INCORPORAR, será criado um código em HTML5, onde poderá ser usado na linha de código e para inserir o vídeo! Basta copiar e colar.

Estilos

Existe 3 metódos de estilos em CSS para ser usados. São eles:

  • CSS Inline - O método CSS Inline, é feito na mesma linha da tag que você deseja.
  • CSS Interno / Local - O método Interno / Local, é feito ná área da tag head, utiliando a tag <style> nela.
  • CSS Externo - O método Externo, consiste na utilização de outro arquivo para configuração das CSSs, além da utilização da tag LINK:CSS dentro da área head.

Esses são os metódos que se possuem, é possível utilizar os três tipos, mas prevalecerá o Inline, em seguida o Local e em terceiro o Externo.
É recomendado o seguinte:

  • Evitar usar o Inline, só se for usar pra algo muito pontual ou específico.
  • O Interno é recomendado utilizar para pequenas configurações nos estilos.
  • O Externo use sem moderação, mas principalmente caso seu site tenha mais de uma página.
