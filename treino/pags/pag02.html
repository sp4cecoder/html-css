Listas

Lista Ordenada

Para a criação de uma lista ordenada, é usado a tag OL, para a sua delimitação, e a tag Li, para a criação dos itens da lista. Exemplo:

  1. Ir para a escola
  2. Voltar para casa
  3. Organizar o planejamento dos RPGs
  4. Mestrar a mesa para o pessoal
  5. Comer
  6. Dormir

Nota: É possível configurar o tipo de marcador da lista, utilizando o parâmetro type na tag OL. Tendo opções para sua utilização:

  • 1
  • A
  • a
  • I
  • i

E com a utilização do parâmetro start com a tag type, é possível dizer em que número/letra começar a lista. Exemplo:

  1. Tomar Banho
  2. Comer pudim
  3. Esovar os dentes
  4. Ir trabalhar
  5. Fazer projeto
  6. Voltar pra casa

Lista Não Ordenada

Para a criação de uma lista não ordenada, é utilizado as tags UL, para a sua delimitação, e a Li, para a criação dos itens da da lista. Exemplo:

  • Açúcar
  • Carne
  • Pão
  • Massa de pastel
  • Mussarela

Nota: É possível configurar o tipo do marcador da lista, através do parâmetro type na tag OL. Tendo opções para sua utilização:

  • Disc
  • Circle
  • Square

Lista de Definição

Funciona como um dicionário, temos os termos e suas descrições, é uma lista sem demarcadores, poderá ser útil para alguns casos.
As tags são as seguintes:

  1. DL - Definition List
  2. DT - Definition Term
  3. DD - Definition Description

Segue um exemplo:

HTML
Linguagem de marcação utilizada para a criação do contéudo de um site.
CSS
Linguagem de marcação utilizada para a criação de design para um site.
JavaScript
Linguagem de programação para a criar interatividades em um site.

Recomendação/Nota: Recomendo, olhar dentro das linhas do arquivo index, para ter uma ideia melhor de como é feito o uso dessas tags.
