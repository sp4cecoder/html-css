Formatações de texto/semântica/entre outros

Negrito / Destaque

Um exemplo claro para deixar em negrito é esse "agora", utilizando a tag b (não semântica).

E podendo deixar em negrito, utiliando a tag STRONG (semântica).

Itálico / Ênfase

Para essa frase, tenho um termo em destaque, utilizando a tag i (não semântica).

E outro termo em destaque, utilizando a tag EM (semântica).

Texto marcado

Pode ser feito marcação em um texto, usando a tag MARK.

E temos outro texto marcado.

Uma coisa interessante, é que é possível fazer uma alteração de cor da marcação usando a tag STYLE, dentro da tag MARK. E também sendo possível usá-la para demarcar a colocação que todas as tags "MARKs" usadas sejam da cor selecionada.

Pra isso é necessário colocar a tag STYLE no corpo do documento, escrever MARK, abrir conchetes e colocar background-color: a cor que deseja, ";" e fecha conchetes. Pronto, a nova cor que a tag MARK tem já está alterada, isso para todas que possam ter em sua linha de código/textos

Caso esteja confuso para poder fazer isso, você pode abrir o arquivo index no visual code e poderá ver o que foi feito aqui. 😎

Texto grande e pequeno

Aqui à um texto grande, usando a tag BIG (obsoleta).

E aqui um texto pequeno, usando a tag SMALL (semântica).

Nota: A tag BIG, pode funcionar ainda, mas já está obsoleta, para fazer alteração dessa forma, acaba sendo necessário o uso de CSS.

Texto deletado e inserido

Pode ser marcado um texto como excluído, para poder indicar que pode ser lido, mas não considerado. Utilizando a tag DEL.

E pode ser marcado um texto como inserido, para dar uma ênfase e poder indicar que ele foi adicionado depois. Utilizando a tag INS.

Nota: Existe também a tag U, que faz também o serviço de sublinhar o texto, mas ele é uma forma não semântica para fazer a sublinhação, então o recomendado é a utilização da tag INS.

Textos subescritos e subscritos

(Subescritos) Para inserir coisas do tipo, que ficaria assim sem a tag: X2+3, já com a tag: X2+3. É utilizado a tag SUP.

(Subscritos) Para inseir coisas do tipo, que ficaria assim sem a tag: H20, já com a tag: H20. É utilizado a tag SUB.

Citações e Códigos

Para fazer citações, é utilizado a tag Q (possui semântica). Mas para citações mais longas (em bloco) e que tenham um parágrafo para si, é utilizado a tag BLOCKQUOTE.
